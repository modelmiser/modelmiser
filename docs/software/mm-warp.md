# mm-warp

Native Wayland remote desktop for COSMIC — 4K H.264 streaming with
keyboard and mouse control.

**Repository:** [github.com/modelmiser/mm-warp](https://github.com/modelmiser/mm-warp)

## What it is

A remote desktop client and server built natively on Wayland protocols,
targeting the COSMIC desktop environment. Streams 4K video via hardware
H.264 encoding with low-latency keyboard and mouse forwarding.
